# gogh RPC

gogh RPC (RIch Presence) allows you to show details about your status when you are on gogh such as which room you are in with how many players

## macOS installation


[Download gogh_rpc.app for macOS](https://github.com/chirpon9/gogh-rpc/releases/download/mac/gogh_rpc.app.zip)

double-click `gogh_rpc.app` to launch it. It will become a background process afterwards.

## Usage

Run the program before or after launching gogh: Focus with Your Avatar. It should change your discord status. Make sure discord is open and activity privacy is turned off.
